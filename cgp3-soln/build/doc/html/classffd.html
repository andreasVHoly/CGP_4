Detailed Description

Free-Form Deformation of geometric models.

Supports Bezier bases with n=1,2,3 that can be set seperately for each dimension.

void ffd::reset ( )

Reset all control points to their initial underformed positions.

These are chosen to ensure that, by linear precision, no deformation happens even if deform is called. In other words, the identity deformation.
